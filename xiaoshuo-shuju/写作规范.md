# 小说写作规范

## 核心原则

### 1. 逻辑一致性(最重要!)
- **必须阅读前3章内容**:写新章节前,必须先读取前面至少3章的内容,了解当前剧情进展
- **时间线连贯**:确保时间、地点、人物状态与前文一致
- **人物记忆**:人物必须记得之前发生的事,不能突然失忆
- **身份一致**:
  - 唐双是明月会的真正老大,老鬼和阿豹都知道并称呼他"唐老大"
  - 不要重复宣布身份或开会介绍
  - 已经发生的事不要重复发生
- **能力连贯**:
  - 已经展示过的能力要有铺垫来源(师傅教的、古籍学的)
  - 不要突然冒出新能力
- **情节衔接**:
  - 上一章结尾在哪里,下一章开头就从哪里继续
  - 不要跳跃时间线或地点而不说明

### 2. 去除AI味
- **避免套话**:
  - ❌ "他深吸一口气"(用太多)
  - ❌ "眼中闪过一丝..."
  - ❌ "嘴角勾起一抹..."
  - ✅ 用更自然的描写
  
- **对话要生活化**:
  - ❌ "你说得对,我明白了"(太正式)
  - ✅ "嗯,懂了"、"行啊"、"得嘞"
  - 加入口头禅、语气词
  - 不同人物说话风格要不同

- **加入生活细节**:
  - 吃饭时的具体菜品、味道
  - 天气、环境的细节
  - 人物的小动作、习惯
  - 手机、电脑等现代元素

- **情绪要真实**:
  - 不要每次都"震惊"、"惊讶"
  - 用更细腻的情绪描写
  - 人物反应要符合性格

### 3. 节奏把控
- **不要水字数**:每一段都要推进剧情或刻画人物
- **对话和描写平衡**:不要全是对话,也不要全是描写
- **冲突要合理**:不要为了冲突而冲突
- **装逼要自然**:不要刻意装,要水到渠成

### 4. 人物塑造
- **性格一致**:
  - 唐双:低调、成熟、有底线
  - 秦雪:干练、关心唐双
  - 室友们:普通大学生,有各自特点
  - 林晓雨:温柔、善良

- **成长轨迹**:人物要有成长,但不能突变

### 5. 修炼体系一致性
- **等级清晰**:
  - 武者:黄阶、玄阶、地阶、天阶
  - 修士:炼气期、筑基期、金丹期、元婴期...
  - 对应关系要准确

- **实力合理**:
  - 唐双是筑基中期,当前时代顶尖
  - 不要随便出现更强的人
  - 突破要有过程和代价

## 写作流程

### 创建新章节前必须做的事:
1. 读取前3章内容(使用huoQuZhangJie工具)
2. 读取800章大纲,确认当前应该写什么
3. 读取相关人物信息
4. 读取相关剧情线
5. 确认时间线、地点、人物状态

### 写作时检查清单:
- [ ] 时间地点与上一章衔接了吗?
- [ ] 人物记得之前的事吗?
- [ ] 对话够生活化吗?
- [ ] 有没有AI味的套话?
- [ ] 有没有生活细节?
- [ ] 逻辑通顺吗?
- [ ] 符合人物性格吗?
- [ ] 符合大纲规划吗?

## 常见错误示例

### ❌ 错误示例1:逻辑断裂
```
第27章结尾:唐双回到学校
第28章开头:唐双在明月会开会
```
**问题**:地点突然变了,没有说明

### ❌ 错误示例2:重复情节
```
第26章:老鬼叫唐双"唐老大"
第28章:秦雪宣布唐双是副会长
```
**问题**:唐双本来就是老大,不需要宣布

### ❌ 错误示例3:AI味重
```
"他深吸一口气,眼中闪过一丝坚定,嘴角勾起一抹笑容..."
```
**问题**:全是套话

### ✅ 正确示例:自然描写
```
唐双靠在椅背上,想了想,笑了:"行,就这么办。"
```

## 特别提醒

1. **见家长情节**:按照800章大纲,应该在121-150章,不要提前
2. **修炼突破**:要有过程,不能一章就突破
3. **新人物出现**:要先创建人物信息,再在章节中使用
4. **剧情线**:重要剧情要记录到剧情线中

## 参考大纲

写作时务必参考 `800章大纲.md`,确保:
- 当前章节在哪个阶段
- 应该推进什么剧情
- 不要跳过重要情节
- 不要提前写后面的情节
